# 张成空间和线性无关

###### 线性组合 （Linear combination）
Def: 对于 $v_1, v_2, ...v_n \in V$, 其线性组合为 $a_1v_1 + a_2v_2 + ... + a_nv_n \in V$ , 其中 $a_1, a_2, ...a_n \in F$

## Intro
- 物理特性用于支持认证“某人拥有某东西” ，但通常要与一个口令或**PIN**结合使用。
- 这种器件应具有存储功能，通常有键盘、显示器等界面部件，更复杂的能支持一次性口令，甚至可嵌入处理器和自己的网络通信设备（如智能卡）。
- 这种器件通常还利用其它密码鉴别方法。

### Integrated Circuit Card
集成电路卡（**Integrated Circuit Card**）简称**IC**卡，其中镶嵌集成电路芯片

**IC**卡的分类
- **IC**卡接口类型
	- 接触式**IC**卡
	- 非接触式**IC**卡
	- 双界面卡
- 嵌入集成电路芯片的形式和类型
	- 非加密存储卡
	- 逻辑加密卡
	- **CPU**卡（又称智能卡）

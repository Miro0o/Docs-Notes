## Cloud Native Configuration Languages
云原生平台配置语言是一种 DSL（Domain Specific Language），通过编程的方式来创建云原生平台，目前已有的云原生 DSL 有：

-   Ballerina
-   Pulumi
-   Cue
-   KCL
-   Kusion

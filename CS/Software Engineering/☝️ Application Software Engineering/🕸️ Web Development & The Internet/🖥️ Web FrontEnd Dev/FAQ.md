## ğŸ‘‰ YAML / JSON /XML
#Web #frontend #yaml #json #xml #config 

YAML(YAL), JSON and XML are all widely used data serialization language.

- YAML stands for â€œ*YAML Aint Markup Language*â€œ.
- JSON stands for â€œ*JavaScript Object Notation*â€œ.
- XML is â€œ*eXtensible Markup Languageâ€* whereas YML is not a markup language.

2. The percentage sets theÂ _opacity_Â (100% being most opaque) ofÂ `$color1`Â on top ofÂ `$color2`.
3. The percentage sets theÂ _transparency_Â (100% being most transparent) ofÂ `$color2`Â on top ofÂ `$color1`.

So to get the color resulting from overlayingÂ `$primary-color`Â at 8% opacity overÂ `$background-color`Â either of the following will get you the same thing (although the first one is more how you were approaching it above):

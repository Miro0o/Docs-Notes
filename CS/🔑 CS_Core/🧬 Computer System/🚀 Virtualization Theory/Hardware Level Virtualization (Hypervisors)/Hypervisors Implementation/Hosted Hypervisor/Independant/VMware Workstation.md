1. 在vmware内查看虚拟机文件位置
2. 进入虚拟机文件，将整个文件夹复制或移动到目标路径。
3. 虚拟机文件夹内的 `.vmx` 文件保存了整个虚拟机的配置信息。待虚拟机文件复制完成后进入vmware软件，选择“打开虚拟机”，然后通过该`.vmx`文件打开虚拟机即可。

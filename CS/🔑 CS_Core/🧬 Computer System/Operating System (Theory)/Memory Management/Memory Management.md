<small>Memory Management Techniques</small>

Fixed /Dynamic Partitioning: 整个程序作为整体，程序全部加载，地址连续
Simple Paging/ Segmentation: 整个程序划分为块，程序全部加载，地址不连续
Virtual Memory Paging/ Segmentation: 整个程序划分为块，程序部分加载，地址不连续

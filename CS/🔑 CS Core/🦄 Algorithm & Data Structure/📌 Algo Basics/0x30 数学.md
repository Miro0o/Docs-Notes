所以为了满足除法运算引入了逆元。

+ ==逆元记为$A^{-1} \pmod P$==，称为A关于P的逆元。仅当$（A,P）=1$ 时，A 存在关于P的逆元。（当X遍历P的一个简化剩余系，AX也遍历P的一个简化剩余系，故一定有一个$X\in(1,P-1)$ 使得$AX \equiv 1 \pmod P$。故需$(A,P)=1$ )

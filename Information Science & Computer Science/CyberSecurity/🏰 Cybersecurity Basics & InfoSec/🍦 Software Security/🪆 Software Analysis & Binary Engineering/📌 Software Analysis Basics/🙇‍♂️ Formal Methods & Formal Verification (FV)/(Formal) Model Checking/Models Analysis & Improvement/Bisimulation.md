### Properties of Bisimulation
**Bisimilarity**
Bisimilarity (largest bisimulation relation) is:
- reflexive
- symmetric
- transitive

**Properties of bisimulations for single transition systems**
Let $R$ be a bisimulation for $T$. Then: 
- R is reflexive
- R is symmetric
- R is transitive

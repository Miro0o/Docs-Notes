## Intro
> VMWare有三个版本
> 1. workstation是单机级，用在个人桌面系统中，需要操作系统支持。
> 2. Server是工作组级，用于服务器，需要操作系统支持。
> 3. ESXi和vSphere是企业级，用于服务器，不需要操作系统支持。ESXi是vSphere的子集，一种裁剪版。

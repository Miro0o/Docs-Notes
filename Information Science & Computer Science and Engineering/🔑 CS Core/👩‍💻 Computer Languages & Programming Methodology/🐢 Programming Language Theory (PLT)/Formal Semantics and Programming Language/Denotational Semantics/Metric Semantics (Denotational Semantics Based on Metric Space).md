## Intro
Metric spaces provide an alternative mathematical foundation for denotational semantics.





> 🤖 GPT-5

度量空间（Metric Space）提供的是另一种结构：
- 元素之间有**距离**
- 可以谈论**收敛**
- 可以用 **Banach 不动点定理** 来处理递归

在这种思路下：
- 程序的意义 = 度量空间中的点
- 程序变换 = 度量空间上的映射
- 递归 = 极限（而不是最小不动点）

这种方法通常被称为：
- **度量语义（Metric Semantics）**
- 或 **基于度量的指称语义**

常见于：
- 并发系统
- 概率程序
- 近似计算
- 语义中的“相似性 / 误差”
